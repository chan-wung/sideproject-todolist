# 투두리스트

Vite + React + TypeScript 기반의 사이드 프로젝트입니다.

## 기술 스택

- **React 18** + **TypeScript**
- **Vite 5**
- **SCSS** (BEM 네이밍, rem 단위)
- **localStorage** 데이터 영속성
- 폰트: Pretendard, GmarketSans

## 기능

- 할 일 추가 / 수정 / 삭제
- 완료 체크 및 완료 항목 일괄 삭제
- 우선순위 설정 (높음 / 보통 / 낮음)
- 마감일 설정 및 기한 초과 강조 표시
- 카테고리 태그 및 필터링
- 상태 필터 (전체 / 진행 중 / 완료)
- 정렬 (최신순 / 우선순위 / 마감일순)
- 삭제 시 확인 모달
- 빈 입력 시 토스트 알림
- 새로고침 후 데이터 유지 (localStorage)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
