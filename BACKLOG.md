## 2. 통계 / 생산성 뷰
주간 완료율, 카테고리별 완료 추이 등을 보여주는 작은 대시보드.
- 기존 `DueSummary` 배너와 톤을 맞춰 같은 디자인 패턴으로 얹을 수 있음

## 3. 전역 키보드 단축키
마우스 없이 주요 기능에 접근.
- `/` — 검색창으로 포커스 이동
- `n` — 새 할일 입력창으로 포커스 이동
- `Esc` — 열린 모달/편집 모드 닫기 (일부는 이미 구현됨)
- `Ctrl/Cmd + Enter` — 할일 입력 중 즉시 저장
- `App.tsx`에 전역 `keydown` 리스너 추가, input/textarea 포커스 중일 땐 무시
