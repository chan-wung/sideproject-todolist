모든 할 일 데이터는 오로지 사용자의 브라우저 내부에만 안전하게 저장되며 외부로 전송되지 않습니다.

## 기술 스택

- **React 18** + **TypeScript**
- **Vite 5**
- **SCSS** (BEM 네이밍, CSS 변수 다크모드, rem 단위)
- **localStorage** 데이터 영속성 (서버 및 DB 없음)
- **PWA** (vite-plugin-pwa) — 홈 화면 설치, 오프라인 실행 지원
- 폰트: Pretendard, GmarketSans

## 주요 기능

- **할 일 관리**
  - 할 일 추가 / 수정 / 삭제 (다중 줄 텍스트 입력 가능, Shift+Enter 지원)
  - 완료 체크 및 완료 항목 일괄 삭제 (완료 탭에서만 표시)
  - 우선순위 설정 (높음 / 보통 / 낮음)
  - 카테고리 태그 (기존 입력 카테고리 자동완성 콤보박스 제공)
  - 할 일 카드에 등록일 · 마감일 표시
  - **할 일 고정(Pin):** 중요한 항목을 목록 최상단에 고정, 파란 테두리로 시각 강조
  - **다중 선택 일괄 처리:** 여러 할 일을 선택하여 카테고리, 우선순위를 일괄 변경하거나 삭제
  - **실행취소(Undo):** 항목 삭제 및 일괄 삭제 시 하단 토스트 알림을 통해 5초 내 복구 가능 (진행 바로 남은 시간 시각화)
  - **반복 할 일:** 매일 / 매주 / 매달 반복 설정, 완료해도 즉시가 아니라 실제 다음 회차 날짜가 되어야 목록에 새로 나타남

- **서브태스크 (하위 항목)**
  - 각 할 일에 세부 체크리스트 추가 / 수정 / 삭제
  - **드래그 핸들로 순서 변경** (네이티브 HTML5 DnD, 라이브러리 없음)
  - 진행도 배지 표시 (예: 2/4)
  - 모든 서브태스크 완료 시 상위 항목도 완료 처리
  - **개별/전체 접기·펴기:** 하위 항목을 항목별 또는 전체 목록 단위로 접어 리스트를 압축 표시 (접힘 상태는 새로고침 후에도 유지, 직접 정렬 시 편의성 향상)

- **필터 · 검색 · 정렬**
  - 상태 탭: **진행 중** (기본값) / 전체 / 완료
  - 마감일 범위: 전체 / 오늘 마감 / 이번 주 마감 / 지연
  - 카테고리별 필터
  - **반복 할 일만 보기:** 반복 주기가 설정된 할 일만 모아서 확인
  - 실시간 텍스트 검색 (제목 · 카테고리)
  - 정렬: **직접 정렬(드래그 앤 드롭)** / 최신순 / 우선순위순 / 마감일순
  - 필터·정렬·검색 상태 새로고침 후 유지 (localStorage)

- **업무 편의성 (UX)**
  - 긴 목록 스크롤 시 화면 상단에 '할 일 추가창' 고정 (Sticky 기능)
  - **스크롤 탑 버튼:** 화면을 아래로 스크롤하면 나타나는 화면 상단 이동 버튼
  - **모바일/태블릿 반응형:** 필터 바 · 메모장 · 마감 요약 배너 세로 스택 배치, 할 일 입력창 상세 옵션(우선순위/마감일/카테고리/반복) 접기·펼치기 토글
  - **퀵 네비게이션:** 현재 할 일 목록을 좌측 사이드바로 표시, 클릭 시 해당 항목으로 스크롤
  - 마감일 설정, 오늘/이번주/지연 마감 요약 배너 알림
  - **마감일 브라우저 알림:** 앱 실행 시 하루 1회, 오늘 마감·지연 건수를 데스크탑 알림으로 안내 (설정에서 On/Off, 옵트인)
  - 다크모드/라이트모드 토글 지원
  - **홈 화면 설치(PWA):** 브라우저 탭이 아닌 독립 앱처럼 설치 가능, 인터넷 연결 없이도 실행

- **빠른 메모장**
  - 멀티 탭 생성 · 제목 편집 · 검색 지원, 자동 저장
  - 탭 고정(Pin) 기능으로 중요 메모 최상단 유지
  - 탭 드래그 앤 드롭으로 순서 변경, 새로고침 후에도 순서 유지
  - **할 일 ↔ 메모 다중 연결:** 할 일 하나에 메모를 여러 개 연결 가능, 수정 모드에서 체크박스로 선택, 메모장에서도 다중 연결·해제 가능, 카드에 연결된 메모 배지로 표시
  - 메모장 사이드바에 각 메모와 연결된 할 일 개수를 배지로 표시

- **공유 · 복사**
  - 할 일/메모를 **클립보드에 복사** (제목 + 하위 항목 체크리스트 텍스트, 완료 항목엔 ✔ 표시)
  - **공유:** Web Share API로 모바일에서 OS 공유 시트 호출, 미지원 환경은 클립보드 복사로 자동 대체
  - **카카오톡 공유:** 별도 팝업 없이 카카오톡 공유창으로 바로 전달
  - **복사(복제):** 할 일/메모를 사본으로 만들어 바로 아래에 추가 (완료 상태·고정 등은 초기화, 템플릿처럼 재사용)
  - 할 일 카드는 클립보드 복사·공유·카카오톡 공유·복사를 **더보기(⋯) 메뉴**로 묶어 액션 버튼 과밀 방지

- **완료 이력**
  - 완료 상태였던 할 일이 삭제될 때(개별 삭제·완료 항목 일괄 삭제 모두 포함) 제목·카테고리·우선순위·하위 항목·완료 시각을 영구 기록
  - 미완료 상태에서 삭제된 항목은 기록되지 않음
  - 날짜별로 그룹핑된 목록으로 조회, 내용·카테고리 텍스트 검색 및 날짜 검색 지원
  - 실행취소(Undo) 시 기록도 함께 원상복구

- **백업 · 설정**
  - 할 일 + 메모 + 완료 이력 데이터를 JSON 파일로 내보내기 / 가져오기 (결과를 색상으로 구분한 토스트 알림: 가져오기 파랑 / 내보내기 초록 / 실패 빨강)
  - 설정 모달에서 할 일 / 메모 / 완료 이력 / 전체 데이터 초기화

- **디자인 시스템**
  - Toss 앱 스타일 기반 UI (카드 그림자, 세그먼트 탭, 필 배지, 중립 색상)
  - `DESIGN.md` — 프로젝트 디자인 토큰·컴포넌트 스펙·Do&Don't 문서화

## 데이터 보호 정책
> **모든 데이터는 사용자의 브라우저 localStorage에만 저장되며, 저장소나 서버로 전송되지 않습니다.** 

## 로컬 실행 가이드 (Windows 사용자)
프로젝트 폴더 내에 있는 **`투두리스트_실행.bat`** 파일을 더블클릭하면 명령어를 입력할 필요 없이 자동으로 로컬 서버가 구동되고 투두리스트 웹 브라우저가 열립니다. (바탕화면에 바로가기를 만들어 사용하시면 편리합니다.)

## 개발자용 명령어

```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev

# 프로덕션 빌드 (TypeScript 컴파일 포함)
npm run build
```
