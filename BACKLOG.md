# 개선 아이디어 백로그

> 검토 후 보류된 항목들. 필요할 때 다시 꺼내서 진행.

## 1. 마감일 브라우저 알림 (Notification API)
오늘 마감 / 지연 항목을 브라우저 데스크탑 알림으로 알려주는 기능.
- `Notification.requestPermission()`으로 사용자 동의 후 사용
- 앱을 안 열어도 알림이 오므로 체감 효과가 가장 큼
- HTTPS(또는 localhost) 환경에서만 동작, 순수 브라우저 기능이라 기존 "데이터는 localStorage에만" 정책과 호환됨

## 2. 통계 / 생산성 뷰
주간 완료율, 카테고리별 완료 추이 등을 보여주는 작은 대시보드.
- 기존 `DueSummary` 배너와 톤을 맞춰 같은 디자인 패턴으로 얹을 수 있음

## 3. 전역 키보드 단축키
마우스 없이 주요 기능에 접근.
- `/` — 검색창으로 포커스 이동
- `n` — 새 할일 입력창으로 포커스 이동
- `Esc` — 열린 모달/편집 모드 닫기 (일부는 이미 구현됨)
- `Ctrl/Cmd + Enter` — 할일 입력 중 즉시 저장
- `App.tsx`에 전역 `keydown` 리스너 추가, input/textarea 포커스 중일 땐 무시

## 5. PWA (홈 화면 설치)
manifest.json + 서비스워커 추가해서 브라우저 탭이 아니라 독립 앱처럼 설치.
- 완전 로컬 저장 방식이라 오프라인 동작과도 궁합이 좋음
