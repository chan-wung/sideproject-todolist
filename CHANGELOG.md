# 변경 이력 (CHANGELOG)

### 26.07.15
- 할 일/메모 클립보드 복사 기능 추가 — 제목 + 하위 항목 체크리스트만 깔끔하게 텍스트로 복사 (완료 항목엔 ✔ 표시, 카테고리·마감일은 제외)
- 할 일/메모 공유 기능 추가 (Web Share API) — 모바일에서 OS 공유 시트 호출, 미지원 환경(주로 데스크톱)은 클립보드 복사로 자동 대체
- 할 일/메모 카카오톡 공유 기능 추가 — 카카오 JS SDK 연동, 텍스트형 템플릿으로 별도 템플릿 등록 없이 공유
- 할 일/메모 복사(복제) 기능 추가 — 원본 바로 아래에 사본 생성, 완료 상태·고정·반복 생성 이력은 초기화하여 템플릿처럼 재사용 가능
- 할 일 카드 액션 버튼이 6개로 늘어나 혼잡해지는 문제를 방지하기 위해 클립보드 복사·공유·카카오톡 공유·복사를 더보기(⋯) 드롭다운 메뉴로 통합
- 아이콘 전용 버튼에 호버/포커스 시 라벨을 보여주는 툴팁 추가 (어떤 기능인지 아이콘만으로 알기 어렵던 문제 개선)
- 완료 이력에도 항목별 및 필터링된 전체 목록 기준 클립보드 복사·공유(Web Share)·카카오톡 공유 기능 추가
- 완료 이력 카테고리 필터를 텍스트 검색과 분리해 셀렉트박스로 변경, 검색창은 내용 텍스트 검색만 담당하도록 축소
- 메모장 탭 목록 아이콘이 과밀해 제목이 잘 보이지 않고 터치 영역이 좁던 문제 수정 — 메모 복사(복제) 기능 제거, 고정/삭제 버튼을 28px 정사각 터치 영역으로 확대
- 메모장 내용 영역도 기능이 늘어나며 복잡해진 문제 개선 — 클립보드 복사·공유·카카오톡 공유를 더보기(⋯) 메뉴로 통합, 연결된 할 일 체크리스트는 접기/펼치기 토글로 기본 접어 공간 확보
- 카카오톡 공유 링크가 개발 중 주소(localhost)로 하드코딩되어 있던 문제 수정 — 실제 배포 주소로 교체
- 완료 이력 날짜 검색을 하루 단위 조회에서 기간(범위) 선택으로 변경 — 달력에서 시작일·종료일을 클릭해 원하는 기간(1주일/1개월 등) 조회 가능
- 완료 이력 복사·공유 텍스트에서 하위 항목 ✔ 표시 제거 — 이력에 남는 항목은 기록 시점에 하위 항목이 항상 전부 완료 상태라 의미 없는 표시였음
- 할 일 입력 폼 순서 변경 — 우선순위·마감일 등 옵션을 입력창보다 위로, 추가 버튼은 힌트 문구와 한 줄로 맨 아래 배치해 작성 흐름에 맞춤 (모바일에서는 버튼 100% 너비)

- 저장 공간 초과 등 localStorage 쓰기 실패 시에도 앱이 중단되지 않도록 예외 처리
- 키보드 접근성 개선 — 마감일 달력, 항목 네비게이션, 메모 탭 목록, 데이터 가져오기를 키보드로 조작 가능
- 스크린리더 지원 개선 — 폼 라벨 연결(우선순위/마감일/카테고리/반복 주기), 검색창·달력 이동 버튼에 이름 부여
- 하위 항목을 드래그할 때 할 일 카드 순서가 함께 바뀌던 문제 수정 (직접 정렬 모드)
- 완료 탭의 '완료 항목 삭제' 버튼 위치를 필터 영역 맨 오른쪽으로 이동
- 토스트 알림에 남은 시간을 보여주는 진행 바 추가 (실행취소 유예 시간 시각화)
- 토스트 진행 바를 실행취소 가능한 토스트에만 표시하도록 조건부 렌더링
- 하위 항목을 전부 체크해 할 일이 자동 완료될 때도 반복 할 일의 다음 회차가 생성되도록 수정 (체크박스 직접 완료와 동일하게 동작)
- 반복 할 일의 다음 회차 생성 시점 수정 — 완료 즉시가 아니라 실제 다음 회차 날짜(매일→다음날, 매주→다음주, 매달→다음달)가 되어야 목록에 나타나도록 변경
- 우선순위 · 카테고리 · 반복주기 뱃지에 동일한 테두리 적용, 색이 없던 반복주기 뱃지에 배경색 부여
- 필터 바에 '반복만 보기' 토글 추가 — 반복 설정된 할 일만 모아서 확인 가능
- 필터 바 2행 구성으로 재배치 — 전체 카테고리/할일 선택/전체 더보기는 왼쪽, 완료 항목 삭제는 오른쪽 끝
- '전체 접기' 버튼에 '할일 선택'과 동일한 활성 색상 적용
- '전체 더보기' 버튼의 화살표 아이콘이 아예 안 보이던 문제 수정 (잘못된 CSS 다중 배경 레이어 문법)
- 모바일에서 '완료 항목 삭제' 버튼 높이가 줄어들던 문제 수정
- **완료 이력** 기능 추가 — 완료 상태였던 할 일이 삭제될 때(개별/일괄 삭제 모두) 제목·카테고리·우선순위·하위 항목·완료 시각을 별도로 영구 기록. 내용·카테고리 검색, 날짜 검색 지원. 툴바 '완료 이력' 버튼으로 조회, 설정에서 이력만 따로 초기화 가능, 백업 내보내기/가져오기에도 포함
