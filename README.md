모든 할 일 데이터는 오로지 사용자의 브라우저 내부에만 안전하게 저장되며 외부로 전송되지 않습니다.

## 기술 스택

- **React 18** + **TypeScript**
- **Vite 5**
- **SCSS** (BEM 네이밍, CSS 변수 다크모드, rem 단위)
- **localStorage** 데이터 영속성 (서버 및 DB 없음)
- **PWA** (vite-plugin-pwa) — 홈 화면 설치, 오프라인 실행 지원
- 폰트: Pretendard, GmarketSans

## 주요 기능

- **할 일 관리**
  - 할 일 추가 / 수정 / 삭제 (다중 줄 텍스트 입력 가능, Shift+Enter 지원)
  - 완료 체크 및 완료 항목 일괄 삭제 (완료 탭에서만 표시)
  - 우선순위 설정 (높음 / 보통 / 낮음)
  - 카테고리 태그 (기존 입력 카테고리 자동완성 콤보박스 제공)
  - **할 일 고정(Pin):** 중요한 항목을 목록 최상단에 고정, 파란 테두리로 시각 강조
  - **다중 선택 일괄 처리:** 여러 할 일을 선택하여 카테고리, 우선순위를 일괄 변경하거나 삭제
  - **실행취소(Undo):** 항목 삭제 및 일괄 삭제 시 하단 토스트 알림을 통해 5초 내 복구 가능

- **서브태스크 (하위 항목)**
  - 각 할 일에 세부 체크리스트 추가 / 수정 / 삭제
  - **드래그 핸들로 순서 변경** (네이티브 HTML5 DnD, 라이브러리 없음)
  - 진행도 배지 표시 (예: 2/4)
  - 모든 서브태스크 완료 시 상위 항목도 완료 처리

- **필터 · 검색 · 정렬**
  - 상태 탭: **진행 중** (기본값) / 전체 / 완료
  - 마감일 범위: 전체 / 오늘 마감 / 이번 주 마감 / 지연
  - 카테고리별 필터
  - 실시간 텍스트 검색 (제목 · 카테고리)
  - 정렬: **직접 정렬(드래그 앤 드롭)** / 최신순 / 우선순위순 / 마감일순
  - 필터·정렬·검색 상태 새로고침 후 유지 (localStorage)

- **업무 편의성 (UX)**
  - 긴 목록 스크롤 시 화면 상단에 '할 일 추가창' 고정 (Sticky 기능)
  - **스크롤 탑 버튼:** 입력창이 Sticky 상태가 되면 자동으로 나타나는 화면 상단 이동 버튼
  - **퀵 네비게이션:** 현재 할 일 목록을 좌측 사이드바로 표시, 클릭 시 해당 항목으로 스크롤
  - 마감일 설정, 오늘/이번주/지연 마감 요약 배너 알림
  - 다크모드/라이트모드 토글 지원
  - **홈 화면 설치(PWA):** 브라우저 탭이 아닌 독립 앱처럼 설치 가능, 인터넷 연결 없이도 실행

- **빠른 메모장**
  - 멀티 탭 생성 · 제목 편집 · 검색 지원, 자동 저장
  - 탭 고정(Pin) 기능으로 중요 메모 최상단 유지
  - 탭 드래그 앤 드롭으로 순서 변경, 새로고침 후에도 순서 유지
  - **할 일 ↔ 메모 연결:** 할 일에 필요한 메모를 연결해 클릭 한 번으로 바로 참고 가능

- **백업 · 설정**
  - 할 일 + 메모 데이터를 JSON 파일로 내보내기 / 가져오기 (결과를 색상으로 구분한 토스트 알림: 가져오기 파랑 / 내보내기 초록 / 실패 빨강)
  - 설정 모달에서 할 일 / 메모 / 전체 데이터 초기화

- **디자인 시스템**
  - Toss 앱 스타일 기반 UI (카드 그림자, 세그먼트 탭, 필 배지, 중립 색상)
  - `DESIGN.md` — 프로젝트 디자인 토큰·컴포넌트 스펙·Do&Don't 문서화

## 데이터 보호 정책
> **모든 데이터는 사용자의 브라우저 localStorage에만 저장되며, 저장소나 서버로 전송되지 않습니다.** 

## 로컬 실행 가이드 (Windows 사용자)
프로젝트 폴더 내에 있는 **`투두리스트_실행.bat`** 파일을 더블클릭하면 명령어를 입력할 필요 없이 자동으로 로컬 서버가 구동되고 투두리스트 웹 브라우저가 열립니다. (바탕화면에 바로가기를 만들어 사용하시면 편리합니다.)

```bash
# 의존성 설치

npm run dev
```
